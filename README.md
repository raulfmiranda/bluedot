# BlueDot Project

Projetos solicitados para a disciplina de Conteúdos Web Mobile pelo prof. Eduardo Mendes.

## 1º Projeto: Framework CSS

### Nome: BlueDot (bluedot.css)

* 3 breakpoints para: desktop,tablet, mobile;
* Layout de grid até 4 colunas
```
Desktop – 4 colunas
Tablet – máx 2 colunas
Mobile – máx 1 coluna
```
* Opção de layout com box ou sem
* Imagens e videos fluidos
* Tabelas sem bordas
* 2 opções de paletas de cores padrão
* Tipografia básica com fontes modernas do GoogleFonts
* Tipografias paras as tags h1 até h6
* Menus horizontais
* Prova de conceito (bluedotPoC.html)
